# Secure Chat System

A real-time chat application with RSA encryption capabilities.

## Features
- Real-time messaging using Socket.IO
- RSA encryption for secure communication
- Clean web interface

## Installation
```bash
cd server
npm install
```

## Usage
```bash
npm start
```

Open http://localhost:3000 in your browser to access the chat system.
